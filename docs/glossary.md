# Глоссарий пользовательского UI

Этот файл фиксирует принятые названия для пользовательских текстов бота. При добавлении экранов, кнопок, подсказок и сообщений сверяйся с ним, чтобы одни и те же сущности назывались одинаково.

| Принятое название | Что означает | Не использовать в UI |
| --- | --- | --- |
| мероприятие | Событие университета, на которое пользователь может записаться: день открытых дверей, экскурсия, занятие, встреча. | ивент, событие |
| Организатор | Пользовательская роль, которая создаёт мероприятия, управляет регистрацией и проверяет записи. | менеджер, администратор, модератор |
| Записан | Статус активной записи пользователя на мероприятие. | подтверждена |

Слово «событие» допустимо в техническом контексте, где речь не о пользовательском объекте записи, а о webhook/update-событиях MAX, YDB или другой инфраструктуры.
